=============
Cloud Storage
=============

Installation
------------

.. code-block:: bash

    pip install cloud-storage


How to use
----------

.. code-block:: python

    >>> from cloud_storage import GoogleCloudStorage
    >>> storage = GoogleCloudStorage()


.. code-block:: python

    >>> from cloud_storage import create_storage_client
    >>> storage = create_storage_client()
